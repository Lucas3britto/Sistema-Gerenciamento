# Sistema gerenciamento para clínica de estética

Foi realizado um estudo em java para dar inicio em um sistema de gerenciamento.
  - Foi iniciado com a inclusão de cadastro para funcionarios,clientes e procedimentos;
  - Dentro dessas funções de cadastro também foi adicionado as pesquisas e deleções de dados dos mesmos;
  - A princípio um sistema bem simples e intuitivo para teste de desenvolvimento em java;
